MY PROJECT

WEB SHORTLINK BEM

The Shortlink Web is a custom URL shortening platform developed by BEM KM Fasilkom Unsri to streamline access to important organizational resources. This tool allows users to convert long and complex URLs into shorter, more userfriendly links. It is primarily used to enhance communication and simplify the sharing of event registrations, official documents, and other digital resources within the university community.

Designed with efficiency and accessibility in mind, the Shortlink Web ensures that users can quickly access relevant information while maintaining a professional and branded appearance for the organization. This project demonstrates the innovative use of technology to support organizational needs and improve the user experience.

WEB PROFILE BEM KM FASILKOM UNSRI

I was actively involved in the development of this website, contributing to both its design and functionality. My role included implementing features to enhance user experience, ensuring seamless navigation, and creating a visually appealing layout that aligns with the organization’s identity.

The Web Profile of BEM KM Fasilkom Unsri serves as the official digital platform representing the Student Executive Board of the Faculty of Computer Science, Sriwijaya University. It provides a comprehensive overview of the organization, including its vision, mission, structure, and ongoing programs.

By offering a user-friendly interface and well-organized content, the Web Profile aims to establish a professional online presence for BEM KM Fasilkom Unsri while fostering engagement and transparency with its audience.

WEB TECHNOLOGY EUPHORIA (TECHPHORIA)

The Web Techphoria is a dedicated platform developed to support Techphoria, a technology-themed event hosted by BEM KM Fasilkom Unsri. The website serves as an information hub for competitions held during the event and provides a seamless registration system for participants.

I played an active role in the development of this website, focusing on both its design and functionality. My contributions included creating a user-friendly interface, implementing registration features, and ensuring that the platform effectively delivered key information about the event and its competitions.

The website was designed to be both visually appealing and efficient, allowing users to easily navigate through event details, competition guidelines, and registration forms. By integrating clear and accessible features, the Web Techphoria successfully enhanced the event’s online presence and streamlined the registration process for participants.

WEB BINA DESA (BINDES)

The Web Bina Desa is a platform created to support the Bina Desa program, aimed at empowering villages by showcasing their profiles and promoting agricultural products and local SMEs. It features a Village Profile section and an E-Commerce System for selling farm products and goods.

I contributed to the development by designing the user interface, integrating the online store, and ensuring seamless functionality. The website enhances the village’s visibility and connects local producers to a wider market, promoting economic growth and sustainability.

WEB BOOK GALLERY

The Web Book Gallery is a platform designed to display detailed information about books, offering users a comprehensive catalog of titles. The website incorporates a CRUD (Create, Read, Update, Delete) system, allowing administrators to manage book listings by adding new titles, updating information, and removing outdated books.

I contributed to the development of this website by implementing the CRUD functionality, designing the user interface, and ensuring smooth navigation for users to easily explore book details. The Web Book Gallery serves as an interactive and efficient platform for managing and showcasing books, improving accessibility for readers and administrators alike.

WEB SIMPLE GALLERY AND NEWS

The Web Simple Gallery and News is a platform designed to showcase a collection of wristwatches and provide news related to the watch industry. The website features a CRUD (Create, Read, Update, Delete) system that allows administrators to manage watch listings and news articles effectively.

As a Full-Stack Developer, I was responsible for both the front-end and back-end development. I implemented the CRUD functionality for adding, updating, and deleting watch details and news posts. The platform offers an easy-to-navigate gallery for users to explore wristwatches and stay updated with the latest news in the watch industry.
